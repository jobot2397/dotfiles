# Starship Configuration

Configuration for the Starship cross-shell prompt.

## Files

- `starship.toml` - Main configuration file

## Setup

Create symlink:

```zsh
ln -s ~/dotfiles/starship/starship.toml ~/.config/starship.toml
```
